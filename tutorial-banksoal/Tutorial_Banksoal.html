Tutorial Aplikasi Belajar SMP - PWA

Apa Itu Aplikasi Ini?

Aplikasi Belajar SMP adalah Progressive Web App (PWA) yang memungkinkan siswa SMP mengatur jadwal pelajaran, mengelola tugas, berlatih soal, dan melacak progres belajar. Semua data disimpan secara lokal di perangkat Anda.

Link Aplikasi: https://vali-tio.github.io/Bank-Soal-Latihan/

Cara Mengakses Aplikasi

1 Buka browser di perangkat Anda (Chrome, Firefox, Safari, dll.)
2 Kunjungi link: https://vali-tio.github.io/Bank-Soal-Latihan/
3 Tambahkan ke layar utama untuk pengalaman terbaik:

Fitur Utama dan Cara Menggunakan

1. üìÖ Jadwal Pelajaran

1 Menambah Jadwal:
  • Pilih hari, mata pelajaran, waktu, dan guru (opsional)
  • Klik "+ Tambah" untuk menyimpan
2 Melihat Jadwal:
  • Jadwal akan ditampilkan per hari dalam bentuk tabel
  • Untuk menghapus: Klik tombol "Hapus" pada jadwal yang ingin dihapus

2. ‚úÖ Manajemen Tugas (To-Do)

1 Menambah Tugas:
  • Isi judul, pilih jenis (PR/Ujian/Catatan), mata pelajaran, tanggal jatuh tempo, dan keterangan
  • Klik "+ Tambah" untuk menyimpan
2 Mengelola Tugas:
  • Centang checkbox untuk menandai tugas sebagai selesai
  • Tombol "Hapus" untuk menghapus tugas

3. üìù Latihan Soal

1 Paket Soal Built-in:
  • Sudah tersedia paket soal Matematika (Persentase, Perbandingan, Luas & Keliling)
2 Memulai Latihan:
  • Pilih paket soal dan jumlah soal (5 atau 10)
  • Klik "Mulai Latihan"
  • Pilih jawaban yang benar pada setiap soal
3 Melihat Riwayat Skor:
  • Riwayat skor latihan sebelumnya ditampilkan dalam tabel

4. üõ† Membuat Paket Soal Kustom

1 Menambahkan Soal:
  • Klik "+ Tambah Soal" untuk menambah soal baru
  • Isi pertanyaan, 4 opsi jawaban, dan pilih jawaban benar dengan radio button
  • Tambahkan pembahasan (opsional)
2 Menyimpan Paket:
  • Beri judul paket dan tag/mata pelajaran
  • Klik "üíæ Simpan Paket"
  • Paket akan muncul di dropdown pilihan paket soal

5. üìä Progres Belajar

1 Ringkasan Statistik:
  • Jumlah tugas aktif dan selesai
  • Jumlah jadwal pelajaran
  • Rata-rata skor latihan
2 Backup Data:
  • Ekspor Data: Download file JSON berisi semua data Anda
  • Impor Data: Unggah file JSON untuk memulihkan data
  • Ekspor PDF: Cetak laporan (perlu integrasi tambahan)
  • Bagikan via WhatsApp: Bagikan laporan melalui WA

6. üîê Fitur Admin

1 Akses Admin:
  • Klik tombol "üîê Admin" di header
  • Jika pertama kali, buat password admin
  • Jika sudah, login dengan password
2 Fitur Admin:
  • Edit/hapus paket soal kustom
  • Kunci/buka paket soal untuk prevent perubahan
  • Hapus semua riwayat skor
  • Ganti password admin

Tips Penggunaan

  • Gunakan Impor/Ekspor untuk backup data secara berkala
  • Kunci paket soal melalui admin jika sudah tidak ingin diubah
  • Manfaatkan filter pada todo list untuk melihat tugas aktif
  • Ganti tema (terang/gelap) dengan klik toggle di header

Dukungan

Pemecahan Masalah

  • Data hilang? Pastikan Anda tidak menghapus cache browser
  • Aplikasi tidak loading? Coba akses ulang atau gunakan browser yang berbeda
  • Lupa password admin? Hapus data browsing untuk aplikasi ini dan buat password baru

Catatan: Aplikasi ini menyimpan semua data secara lokal di perangkat Anda. Data tidak dikirim ke server mana pun.
